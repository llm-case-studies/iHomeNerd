# Claude Design HTML Exports

This directory contains extracted browser-runnable artifacts from Claude
Design.

## What works here

- `index.html` gives a simple launch page
- `ui_kits/ui_kits/android_node/index.html` previews the Android node-class app
- `ui_kits/ui_kits/ios_controller/index.html` previews the iOS controller app
- `ui_kits/ui_kits/command_center/index.html` previews the dense web Command Center
- `design_system_preview/preview/` holds supporting token and component previews

## What these files are

These are design/reference artifacts:
- React-in-browser prototypes
- generated component code
- preview HTML

They are useful for:
- implementation reference
- visual QA
- extracting layout and component ideas

They are not yet the production Android or web code.
