# Initiative Testing

This folder holds validation artifacts organized by initiative and sprint slug.

Use this shape for new product work:

```text
testing/initiatives/<initiative>/<sprint-slug>/
  request.md
  result.md
  evidence/
```

The matching expert sprint pack should live under:

```text
docs/expert-briefs/initiatives/<initiative>/<status>/<sprint-slug>/
```

Legacy mobile-first request/result files may remain under `mobile/testing/`
until they are migrated.
