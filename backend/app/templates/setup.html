🦊 Extra step for Firefox

Firefox uses its own certificate store and ignores your system certificates by default. Choose one:

  1. Recommended: Open Firefox, type about:config in the address bar, search for security.enterprise_roots.enabled, and set it to true. This makes Firefox trust your system certificates (including iHomeNerd's). One-time change — persists across restarts.
  2. Alternative: In Firefox go to Settings → Privacy & Security → Certificates → View Certificates → Authorities → Import → select /tmp/ihomenerd-ca.crt → check "Trust to identify websites" → OK.
